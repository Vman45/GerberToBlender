◆ harden()

def import_pcb.harden ( )

Duplicates all component layers in the pcb and joins them into a single object.

It then moves this object out of the layers collection and into the primary collection. The single board is placed at the origin with a geometry-centralized local origin and the layered board is moved off to the side

def import_pcb.import_svg (   dir,
  file 
)

Brings in the SVG file, applies the x and y orientation, converts the curves to meshes, scales it to 1 meter in blender equals 1 millimeter in the real world, and places the objects into a collection ...

(still to come: extrusions, height placement, cut the holes, and join a copy into a completed version)
Uses Blender 2.8.2 or higher API

Parameters
dir-the directory where the files are located
file-the list of SVG files representing the Gerber Files / PCB
